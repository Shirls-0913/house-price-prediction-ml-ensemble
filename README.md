# House Price Prediction using Machine Learning

## Overview
This project predicts house prices based on various property features using machine learning regression models. It includes data preprocessing, outlier handling, model training, hyperparameter tuning, and evaluation.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, SciPy

## Key Features
- Data preprocessing and cleaning  
- Outlier handling using Winsorization  
- Feature scaling using StandardScaler  
- Model building using ensemble regression techniques  
- Hyperparameter tuning using RandomizedSearchCV  
- Cross-validation for robust performance evaluation  
- Model evaluation using RMSE and R² score  

## Models Used
- Random Forest Regressor  
- Gradient Boosting Regressor  

## Workflow
1. Data preprocessing and outlier treatment  
2. Exploratory data analysis (EDA)  
3. Train-test split  
4. Feature scaling  
5. Model training and tuning  
6. Cross-validation  
7. Model evaluation  

## Results
Ensemble models such as Random Forest and Gradient Boosting provide strong performance in predicting house prices with good generalization.
